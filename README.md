# Salary Prediction Model

## Overview

Built a Machine Learning model to predict employee salaries based on features such as experience, job role, and other relevant factors.

## Tasks Performed

* Data Collection and Preprocessing
* Feature Selection
* Linear Regression Model Training
* Salary Prediction
* Model Evaluation using Mean Squared Error (MSE)

## Tools Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## Outcome

Developed a Linear Regression model capable of predicting salaries and evaluated its performance using MSE.
